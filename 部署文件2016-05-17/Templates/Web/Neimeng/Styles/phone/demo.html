第一步：使用font-face声明字体 
@font-face {font-family: 'iconphone';
    src: url('iconphone.eot'); /* IE9*/
    src: url('iconphone.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
    url('iconphone.woff') format('woff'), /* chrome、firefox */
    url('iconphone.ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
    url('iconphone.svg#iconphone') format('svg'); /* iOS 4.1- */
}
第二步：定义使用iconphone的样式

    font-family:"iconphone" !important;
    font-size:16px;font-style:normal;

    -webkit-text-stroke-width: 0.2px;
    -moz-osx-font-smoothing: grayscale;}
