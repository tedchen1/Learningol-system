第一步：使用font-face声明字体 
@font-face {font-family: 'icondepart';
    src: url('icondepart.eot'); /* IE9*/
    src: url('icondepart.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
    url('icondepart.woff') format('woff'), /* chrome、firefox */
    url('icondepart.ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
    url('icondepart.svg#icondepart') format('svg'); /* iOS 4.1- */
}
第二步：定义使用icondepart的样式 
.icondepart{
    font-family:"icondepart" !important;
    font-size:16px;font-style:normal;
    -webkit-font-smoothing: antialiased;
    -webkit-text-stroke-width: 0.2px;
    -moz-osx-font-smoothing: grayscale;}
